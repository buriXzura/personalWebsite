MassTree-PMEM

A High-Performance Persistent Memory B-link Tree with Hierarchical Concurrency Control
Rahul Prajapat
University of Sydney, School of Computer Science

Abstract

We present MassTree-PMEM, a novel persistent memory (PMEM) optimized B-link tree implementation that addresses critical limitations in existing concurrent index structures. Our design introduces a hierarchical two-tier locking mechanism that enables lock-free reads and concurrent writes while maintaining crash consistency. MassTree-PMEM resolves atomicity issues present in the RECIPE framework's structural modification operations and demonstrates superior performance characteristics compared to DRAM-based implementations.
